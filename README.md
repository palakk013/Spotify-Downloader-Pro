# Spotify Downloader Pro

A Spotify Downloader web application built using HTML, CSS and JavaScript.

## Features

- Spotify-inspired UI
- Search Spotify song
- Fetch song information
- Album cover
- Artist details
- Download button
- Responsive design
- RapidAPI integration

## Technologies Used

- HTML5
- CSS3
- JavaScript
- RapidAPI

## Installation

1. Clone this repository.
2. Open config.js
3. Add your RapidAPI key.
4. Open index.html using Live Server.

## Note

This project uses a third-party RapidAPI service. Download availability depends on the API provider.
